Looper commands

What's this?

Basically for a video on how to loop something extremely long (and also high quality, e.g. Yoshi 1 hours). I used the command line to do the looping, but I think there are some people who want to do this, and may not be familiar with the command line (and I hate it when they don't know how to use tech). So I made this exact tool that "generates" a command that would help the user loop a video. i also made this at 2 am it's fine

do note that this will only work with the most common bash and zsh, so if you're on windows, you might want to write it into a file.
