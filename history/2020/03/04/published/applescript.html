Here is where I'll share my AppleScripts:

Random Volume

set originalv to output volume of (get volume settings)
set repeatz to the text returned of (display dialog "How many times do you want to set the volume randomly?" default answer "100")
repeat repeatz times
	(set volume output volume random number from 0 to 100)
end repeat
set volume output volume originalv
Typing spammer (used for texting):
set spamText to the text returned of (display dialog "What to spam?" default answer "")
set repeatTimes to the text returned of (display dialog "Times to spam" default answer "50")
delay 5
tell application "System Events"
	repeat repeatTimes times
		keystroke spamText
		key code 36
	end repeat
end tell
