Q: How is this made?
A: This was inspired by the Real Time HTML editor by someone else who had alread made. You can find it here.

Q: Why do I need to enable JavaScript to run?
A: This was originally created on JavaScript, and the only way to change elements in HTML is using JavaScript.

Q: How are the files saved?
A: The editor heavily relies on local storage. Clearing your local storage will clear all text entered on the editor, and switching to another browser shows an empty box instead. Text you entered on one site does not transfer to another site. You can now also save the HTML code as file.

Q: Why does tab indenting not work?
A: I have been attempting to allow tab indenting, but it seems like that the methods I have been trying does not work.

Q: What are the limitations of this live editor?
A: Limitations:

  • It only works with HTML
  • You can't open files
  • The data only gets saved on to the specific site. If you transfer to another website, it's not saved.
  • Tab indenting doesn't work
  • No Syntax Highlighting (better remember those syntax)
  • No code suggestions

    • Q: When does the editor save the changes?
    A: After refreshing which is after 750 MS of inactivity only after a key release.

    Q: Why would you create a save and open feature if you already have a way to save automaticlly?
    A: The open from file and save to file is used for some who need to save the HTML to an actual file.
