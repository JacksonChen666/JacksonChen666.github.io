Q: How is this made?
A: This was inspired by the Real Time HTML editor by someone else who had alread made. You can find it here.

Q: Why do I need to enable JavaScript to run?
A: This was originally created on JavaScript, and the only way to change elements in HTML is using JavaScript.

Q: How are the files saved?
A: The editor heavily relies on local storage. Clearing your local storage will clear all text entered on the editor, and switching to another browser shows an empty box instead. Text you entered on one site does not transfer to another site.

Q: Why does tab indenting not work?
A: I have been attempting to allow tab indenting, but it seems like that the methods I have been trying does not work.

Q: What are the limitations of this live editor?
A: Limitations:

  • It only works in HTML style (HTML, JS, and CSS can be combined into one HTML file)
  • You can't open files
  • You can't save it to a file
  • The data only gets saved on to the specific site. This is also affected locally.

    • Q: When does the editor save the changes?
    A: When you release any key on your keyboard.
    Q: Isn't that why it is causing the screen flashes?
    A: no. The file is only saved after refreshing the live view. Refer to 'refresh()' in the code.
