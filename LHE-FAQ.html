Q: How is this made?
A: This was inspired by the Real Time HTML editor by someone else who had already made. You can find it here.

Q: Why do I need to enable JavaScript to run?
A: This was originally created on JavaScript. There's nothing else within HTML that I can use to even replicate this.

Q: How are the files saved?
A: Local storage. Clearing it will remove any text saved. This also means switching between devices won't be the same.

Q: What are the limitations of this live editor?
A: Limitations:

  • You can only edit HTML
  • The data only gets saved on to the specific site. If you transfer to another website, it's not saved unless you open a file.
  • No Syntax Highlighting (better remember those syntax)
  • No code suggestions

    • Q: When does the editor save the changes?
    A: After refreshing which is after 750 MS of inactivity only after a key release.

    Q: The editor on the site I was using removed the editor and I have important stuff on there! What do I do?
    A: This might not be easy. Go to the site, open inspect element (iPhone and Safari, Android Chrome), application, local storage, and copy the text from the "RTE_HTML" key. (Remember, the data is most likely save only on one browser!)
